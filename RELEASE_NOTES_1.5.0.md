# Dicode Config Checker v1.5.0 — Stable

## سریع‌تر، با همان کیفیت

- تعداد بررسی‌های هم‌زمان در Windows و Linux از ۴ به ۸ افزایش یافت.
- Android از صف‌های موازی بزرگ‌تر برای TCP و Xray استفاده می‌کند.
- تعداد تلاش‌ها، حداقل موفقیت و رتبه‌بندی بر پایهٔ میانهٔ تأخیر تغییری نکرده‌اند؛ سرعت بیشتر شده، نه معیار قبولی آسان‌تر.

## اصلاح فوری دریافت کانال

- با لمس «شروع دریافت»، دریافت کانال‌ها بلافاصله شروع می‌شود؛ آماده‌سازی Xray در پس‌زمینه انجام می‌شود و جلوی مرحلهٔ اول را نمی‌گیرد.
- پاسخ‌های خالی یا صفحه‌های خطای `t.me` که با کد ۲۰۰ برمی‌گردند دیگر موفق تلقی نمی‌شوند؛ برنامه همان کانال را خودکار از `telegram.me` می‌خواند.
- اگر فهرست کانال‌های تنظیمات خالی باشد، برنامه آن را تشخیص می‌دهد و از پاک‌کردن ناخواستهٔ فایل کانال‌ها جلوگیری می‌کند.

## راه‌اندازی مرحله‌ای ساب GitHub

- در تنظیمات یک دکمهٔ مستقیم برای باز کردن صفحهٔ صحیح ساخت GitHub Classic PAT اضافه شد.
- آموزش سه‌مرحله‌ای داخل برنامه، از ساخت توکن تا چسباندن و فعال‌سازی، برای کاربران بدون تجربهٔ GitHub نمایش داده می‌شود.
- دسترسی لازم فقط `public_repo` است.
- انتشار در خطاهای موقت DNS/شبکه چند بار خودکار تلاش می‌شود؛ اگر شبکه در دسترس نباشد خروجی در صف می‌ماند و بدون گرفتن تست دوباره، خودکار مجدداً منتشر می‌شود.
- برای لینک ریپازیتوری، `sub.txt` و `proxy.txt` آیکون کپی و کپی با لمس خود لینک افزوده شد.
- پس از ذخیرهٔ توکن، دکمهٔ «ساخت / اتصال ریپازیتوری ساب» ریپازیتوری عمومی را همان لحظه ایجاد یا به ریپوی قبلی متصل می‌کند؛ لازم نیست برای اولین تست صبر شود.
- در تب کانفیگ و تب پروکسی دکمهٔ مستقل «به‌روزرسانی ساب» افزوده شد. پس از انتشار، وضعیت دقیق «منتشر شد» نمایش داده می‌شود؛ اگر محتوای همان فایل قبلاً روی GitHub بوده باشد، برنامه «تکراری است» اعلام می‌کند و ارسال دوباره انجام نمی‌شود.
- این کنترل‌ها و تشخیص محتوای تکراری در Android، Windows و Linux یکسان هستند.

## کانال‌ها و خروجی

- فهرست پیش‌فرض به ۳۲۴ کانال به‌روزرسانی و در Windows، Linux و Android یکسان شد.
- فایل `channels.txt` در نسخه‌های دسکتاپ همراه برنامه بسته‌بندی می‌شود تا اولین اجرای نسخهٔ portable نیز همین فهرست را دریافت کند.
- `sub.txt` و `proxy.txt` اختصاصی همچنان بعد از هر تست موفق به‌روزرسانی می‌شوند.
