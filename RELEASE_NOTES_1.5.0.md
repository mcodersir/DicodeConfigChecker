# Dicode Config Checker v1.5.0 — Stable

## سریع‌تر، با همان کیفیت

- تعداد بررسی‌های هم‌زمان در Windows و Linux از ۴ به ۸ افزایش یافت.
- Android از صف‌های موازی بزرگ‌تر برای TCP و Xray استفاده می‌کند.
- تعداد تلاش‌ها، حداقل موفقیت و رتبه‌بندی بر پایهٔ میانهٔ تأخیر تغییری نکرده‌اند؛ سرعت بیشتر شده، نه معیار قبولی آسان‌تر.

## راه‌اندازی مرحله‌ای ساب GitHub

- در تنظیمات یک دکمهٔ مستقیم برای باز کردن صفحهٔ صحیح ساخت GitHub Classic PAT اضافه شد.
- آموزش سه‌مرحله‌ای داخل برنامه، از ساخت توکن تا چسباندن و فعال‌سازی، برای کاربران بدون تجربهٔ GitHub نمایش داده می‌شود.
- دسترسی لازم فقط `public_repo` است.

## کانال‌ها و خروجی

- فهرست پیش‌فرض به ۳۲۴ کانال به‌روزرسانی و در Windows، Linux و Android یکسان شد.
- فایل `channels.txt` در نسخه‌های دسکتاپ همراه برنامه بسته‌بندی می‌شود تا اولین اجرای نسخهٔ portable نیز همین فهرست را دریافت کند.
- `sub.txt` و `proxy.txt` اختصاصی همچنان بعد از هر تست موفق به‌روزرسانی می‌شوند.
